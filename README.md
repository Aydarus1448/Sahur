#**flesystem.c**
файловая система

1. `open_fs_file("имя_файла")`  
Открывает файл. Если его нет - создаёт.  

2. `view_file(fs, "имя_файла")`  
Показывает содержимое файла. Просто выводит на экран.

3. `delete_file(fs, "имя_файла")`  
Удаляет файл из системы.
